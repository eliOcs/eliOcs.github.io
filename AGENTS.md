# eliocapella.com

- Static site: edit HTML and CSS directly. Do not introduce frameworks, a build step, or JavaScript unless explicitly requested.
- Target modern browsers. Prefer modern HTML and CSS best practices over legacy browser compatibility workarounds.
- Local development: `npm start`
- Tests: `npm test`
- Shared styles live in `/style.css`.
- Pages use directory-based `index.html` files. Blog posts live in `/blog/<slug>/index.html`; keep post images in the same folder.
- Preserve the current priorities when editing: semantic HTML, accessibility, readability, and fast loading.
